# Design System Narrative: Brutalist Precision

## Vision and Intent
This design system is built for clarity and impact, utilizing a high-contrast palette and sharp geometric forms. It emphasizes a technical, utility-focused layout while maintaining a strictly angular aesthetic.

## Color Strategy
The system operates in a **dark mode** environment, leveraging a sophisticated industrial palette:
- **Primary Intent**: A vibrant, high-energy purple (#5836fd) serves as the core interactive color for CTAs and critical UI signals.
- **Secondary Support**: A muted, desaturated lavender (#756bc8) provides a professional counterpoint for secondary actions and subtle depth.
- **Tertiary Accent**: A deep, earthy orange (#972200) is used sparingly for highlights that require immediate visual distinction.
- **Neutral Foundation**: A mid-tone gray (#787583) replaces pure black, creating a softer yet still distinctly industrial surface for backgrounds and structural containers.

## Typography
The typographic system emphasizes readability and technical precision:
- **Headlines and Body**: The system uses **Inter**, a versatile sans-serif designed for screen clarity, ensuring that both high-level messaging and long-form content remain legible.
- **Labels and Metadata**: **Space Grotesk** provides a technical, monospaced-adjacent feel for micro-copy, UI labels, and data points, reinforcing the system's geometric DNA.

## Geometry and Spacing
- **Corner Treatment**: The design adheres to a **Sharp (0)** roundedness profile. Every button, card, and input field features 90-degree angles, projecting a brutalist, no-nonsense architectural vibe.
- **Spatial Rhythm**: With a **Normal (2)** spacing setting, the layout strikes a balance between information density and breathing room. It is structured for efficiency, remaining approachable for complex dashboards or functional applications.
